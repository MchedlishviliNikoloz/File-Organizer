# 📁 File Organizer

მარტივი და ძლიერი command-line პროგრამა, რომელიც ავტომატურად ალაგებს ფაილებს ნებისმიერ ფოლდერში ნაგულისხმევი, ფაილის ტიპის ან თქვენი საკუთარი კონფიგრუაციის მიხედვით — ასევე აქვს დუპლიკატების პოვნის და undo ფუნქცია.

🇬🇪 ქართული | 🇺🇸 [English](README.md)
---

## ✨ ფუნქციონალი

- 📂 **ნაგულისხმევი კატეგორიზაცია** — ფაილები დალაგება წინასწარ განსაზღვრულ კატეგორიებში (სურათები, დოკუმენტები, აუდიო, ვიდეო, კოდი, არქივები)
- 🔤 **გაფართოებით კატეგორიზაცია** — ფაილები იჯგუფება გაფართოების მიხედვით (`.jpg`, `.pdf` და სხვა)
- ⚙️ **კონფიგურაციის მართვა** — შეგიძლია შეცვალო და გააფართოო ნაგულისხმევი წესები
- 👤 **პირადი კონფიგურაცია** — შექმენი სრულიად საკუთარი კონფიგურაცია
- 🔍 **დუბლიკატების პოვნა** — MD5 hash-ით პოულობს და გადაიტანს იდენტური კონტენტის ფაილებს
- ↩️ **გაუქმება (Undo)** — ნებისმიერ დროს შეგიძლია გააუქმო ბოლო ორგანიზება
- 📊 **Progress bar** — რეალურ დროში გიჩვენებს გადატანის პროგრესს
- 📝 **ლოგირება** — ყველა ოპერაცია იწერება `organizer.log` ფაილში
- 🔄 **საწყის კონფიგურაციაზე დაბრუნება** — ნებისმიერ დროს შეგიძლია აღადგინო ორიგინალი კონფიგურაცია

## 📋 მოთხოვნები

- Python 3.10+
- tqdm

---

## 🚀 ინსტალაცია

**1. გადმოწერე პროექტი:**
```bash
git clone https://github.com/MchedlishviliNikoloz/file-organizer.git
cd file-organizer
```

**2. დააინსტალირე dependencies:**
```bash
pip install tqdm
```

**3. გაუშვი პროგრამა:**
```bash
python main.py
```

## 📖 გამოყენება

გაუშვი პროგრამა და შეიყვანე ფოლდერის მისამართი:

```
📂 Enter folder path: /Users/username/Desktop/MyFolder

1. Organize by default categories
2. Organize by file type
3. Organize by my categories
4. Find and move duplicates
5. Manage default categories
6. Manage my categories
7. Undo last organize
```

### ნაგულისხმევი კატეგორიები

ფაილების ორგანიზება შემდეგ ფოლდერებში:

| კატეგორია | გაფართოებები |
|-----------|-------------|
| 📷 images | .jpg, .jpeg, .png, .gif, .webp, .svg, .bmp ... |
| 📄 documents | .pdf, .docx, .txt, .xlsx, .pptx, .md ... |
| 🎵 audio | .mp3, .wav, .flac, .aac, .ogg ... |
| 🎬 video | .mp4, .mov, .avi, .mkv, .wmv ... |
| 💻 code | .py, .js, .html, .css, .ts, .json ... |
| 🗜️ archives | .zip, .rar, .7z, .tar, .gz |
| 📦 other | ყველა დანარჩენი |

### უკან დაბრუნება (Undo)

ორგანიზების შემდეგ ნებისმიერ დროს შეგიძლია გააუქმო ბოლო ოპერაცია — აირჩიე **7. Undo last organize**. ისტორია ინახება `.undo_log.json` ფაილში.

### დუბლიკატების პოვნა

პროგრამა სკანირებს ფაილებს **MD5 hash**-ის გამოყენებით და პოულობს იდენტური კონტენტის ფაილებს (სახელის მიუხედავად). დუბლიკატები გადავა `duplicates/` ფოლდერში, ორიგინალი ფაილი კი თავის ადგილზე რჩება.
